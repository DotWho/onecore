Imgup 实例：

单个，限制大小
HTML
```` <div data-toggle="imgup"> <input type="hidden" data-type="imgup" value=""> </div> ````
通过 data 限制大小(需带单位KB,MB,GB)
```` <div data-toggle="imgup" data-size="2MB"> <input type="hidden" data-type="imgup" value=""> </div> ````
通过 JavaScript 调用
```` $('.imgupx').imgup({ max: 5, size: '4MB' }); ````
