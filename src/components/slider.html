Slider 实例：

HTML
```` <div class="slider" data-toggle="slider"> <div> <a href="#" style="background-image: url('../img/1.jpg')"></a> <a href="#" style="background-image: url('../img/2.jpg')"></a> <a href="#" style="background-image: url('../img/3.jpg')"></a> </div> </div> ````
通过 data 属性
```` <div data-auto="true" data-arrow="true" ... > ... </div> ````
通过 JavaScript 调用
```` $el.slider({ auto: true, arrow: true, stop: 5000, speed: 600, ease: 'ease' }); ````
