Select 实例：

开启自定义select
HTML
```` <select data-toggle="select"> <option value="1">选项1</option> <option value="2">选项2</option> <option value="3">选项3</option> </select> ````
通过 data 设置默认选中值
```` // 开启自定义select 并设置选中值 <select data-select="2"> <option> ... </option> <option> ... </option> </select> ````
通过 JavaScript 调用
```` $el.select({'select': '2'}); // 更新select $el.select('update'); ````
